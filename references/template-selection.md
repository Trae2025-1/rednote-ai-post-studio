# Template Selection

Choose templates by the user's uploaded logo count and content type. Do this before designing.

## Decision Tree

1. Count the uploaded logo assets first. Logo count is authoritative.
2. If the user mentions PDF, file, resource pack, checklist, manual, or guide download: use **PDF Resource Template**.
3. Else if exactly two logos are uploaded, or the user clearly supplies two distinct tool assets: use **Two-Logo Workflow Template**.
4. Else if the user asks for "建议收藏", "不会用", "隐藏技巧", "完整教程", or a punchy save-worthy post: use **Suggested-Save Poster Template**.
5. Else if the slogan naturally supports three small benefits: use **Three-Benefit Clean Template**.
6. Else use **Single-Logo Tutorial Template**.

Before finalizing the template, check title fit:

- The selected reference must keep connected Chinese title phrases intact.
- Do not choose a reference if it forces the tool name into the middle of a phrase.
- Bottom benefits or CTA text must fit the reference's designed chip, pill, band, or label treatment.
- If the title is a complete sentence or phrase, prefer a layout that can show it as one phrase plus a separate product/logo area.

## Two-Logo Workflow Template

Use when there are exactly two logos or two named tools.

Do not choose this template merely because the copy contains terms such as "搭建", "工作流", "自动化", or "结合". One uploaded logo still uses a one-logo template.

Reference folders:

- `/Users/trae/Documents/Trae/资料/头图案例库/两个logo/`

Allowed logo arrangements:

- Two large icons side-by-side with `+` or `x`.
- Two icons at the lower half after large title block.
- Two small logos at top with divider, then text-heavy body.

Keep both logos original. Do not merge, redraw, recolor, or stylize either logo.

The PNG must visually include the original logo. If the logo is missing after export, fix the export and regenerate before responding.
