# Compliance Rules

## Hard Avoid

Do not use:

- 保证涨粉
- 保证变现
- 月入几万
- 暴涨
- 躺赚
- 稳赚
- 全网最强
- 爆款公式
- 永久免费
- 私信领取
- 加我领取
- 留联系方式
- 微信/群聊/网盘导流
- 夸大工具效果
- 暗示平台外交易

## Use Carefully

Soften or avoid:

- 资料包
- 免费领取
- 自动赚钱
- 变现秘籍
- 必看
- 一键搞定一切
- 小白闭眼冲

## Safer Alternatives

Use:

- 我整理了一版
- 可以作为参考
- 适合新手先试
- 实测下来比较稳
- 这套流程可以复用
- 评论区可以交流
- 后续继续补充
- 我下次继续测

## Interaction Guidance

Avoid:

- 评论"AI"领取资料
- 私信我发你模板
- 加我进群

Prefer:

- 需要的话，我后面继续整理成一版流程。
- 你们更想看哪个工具的实测，我下次继续测。
- 有类似场景也可以评论区交流。

## Final Pass

Before final output, check:

- Any exaggerated promise?
- Any forced traffic diversion?
- Any sensitive domain advice?
- Any misleading tool capability?
- Any overconfident claim without evidence?
